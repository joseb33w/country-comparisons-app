# CountryScope

CountryScope is a polished browser app for comparing a simple 5-country live data set and seeing a small personal bio comparison alongside the country data.

## Features

- Creative tabbed atlas interface instead of one long scrolling page
- Simple live pull for 5 countries only: United States, Canada, Mexico, Japan, and Brazil
- Search and add countries from the simplified live set to a comparison board
- Uses live REST Countries API data in the browser
- Compare population, area, density, region, languages, currencies, capitals, and more
- Personal bio card that compares your home country/languages/interests with selected countries
- Sortable comparison cards, visual metric bars, and a detailed comparison table
- Responsive, mobile-friendly design

## Preview

Open `index.html` in a browser or enable GitHub Pages for this repository.

## Data source

Country data is loaded client-side from the public REST Countries API using a small 5-country request instead of the full global country list.
